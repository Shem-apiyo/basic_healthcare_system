# Basic Management System (C)

A simple console-based management system written in C to show fundamental skills.  

## Features
- Menu-driven console interface
- Data input and display
- Simple record file management
- Written entirely in C


## How to Compile and Run

1. Clone the repository:
   git clone https://github.com/Shem-apiyo/basic_healthcare_system/.git

2. Navigate into the project directory:
   cd basic_healthcare_management_system

3. Compile the program:
   gcc .c -o management_system

4. Run the program:
   ./management_system

## Project Structure
- `basic_healthcare_management_system.c` — main source code file
- `README.md` — project documentation

## Purpose
This project was created as a learning exercise c programming skills:
- C syntax and control structures
- User input handling
- Console-based application design
